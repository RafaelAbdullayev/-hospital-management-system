Программа для регистрации и входа пользователей

Эта программа представляет собой простое приложение для регистрации пользователей и входа в систему. В программе есть функционал для регистрации и входа пользователей, а также для управления пользователями администраторами.

Особенности программы

Регистрация Пользователи могут зарегистрироваться, указав свое полное имя, имя пользователя, пароль, адрес электронной почты и роль (админ или пользователь).
Вход**: Зарегистрированные пользователи могут войти в систему, используя имя пользователя и пароль.
Панель администратора Администраторы могут управлять данными пользователей, например, удалять или обновлять их.
Управление пользователями Администраторы могут просматривать и управлять всеми зарегистрированными пользователями.

Функциональность программы

1. Регистрация (Sign Up)
   Новый пользователь вводит свое полное имя, имя пользователя, пароль, адрес электронной почты и роль (админ или пользователь).
   После успешной регистрации данные сохраняются, и пользователь может перейти к другим экранам программы.

2. Вход (Sign In)
   Пользователь может войти в систему, используя имя пользователя и пароль, указанные при регистрации.
   После входа будет отображаться имя пользователя и его роль.

3. Панель администратора
   Администраторы могут управлять данными пользователей. Можно добавить новых пользователей или изменить существующих.
   Только администраторы могут управлять всеми пользователями.

4. Навигация
   Пользователи могут возвращаться на предыдущие экраны с помощью кнопки Back (Назад)
   
Python 3.13
Tkinter (для создания графического интерфейса)
Pickle (для хранения данных пользователей)
Хранение данных
Программа сохраняет данные пользователей в файл users.pickle. Этот файл обновляется при добавлении новых пользователей. В случае потери этого файла данные пользователей будут утеряны, поэтому рекомендуется сохранять файл в безопасном месте.
Понимание кода
В этой программе используется библиотека Tkinter для создания графического интерфейса пользователя. Для хранения данных используется библиотека Pickle
